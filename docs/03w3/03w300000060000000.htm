Upgrade apps

If you want to upgrade apps created in the previous version of Web AppBuilder (Developer Edition) to a newer version, you can achieve it with a command utility.

Steps:
  1. Stop all running instances of Web AppBuilder (Developer Edition) if any.
  2. Open Command Prompt window.
  3. Browse to the \server directory of your newer version of Web AppBuilder Installation.
  4. Enter node upgrade < your_previous_version_Web_AppBuilder_installation_directory > command , and press Enter key.

    You previous version of Web AppBuilder installation directory can be something like C:\arcgis-web-appbuilder-1.0.

  5. Once upgrade completed, start the newer version of Web Appbuilder to check the result. You should see all the upgraded apps listed in the table.

    If upgrade failed, see upgrade.log file in \server\logs directory of Web AppBuilder installation.

Running the command multiple times will upgrade and import apps multiple times into the newer version of Web AppBuilder. This is by design.

If you have custom widgets and themes in the app created in the previous version of Web AppBuilder, the command utility will import them into the newer version.
